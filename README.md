# Spring-Cafeteria

| Column Name      | Data Type                                     | Constraints             |
|------------------|----------------------------------------------|-------------------------|
| `id`             | `bigint`                                     | NOT NULL, PRIMARY KEY   |
| `nickname`       | `varchar(30)`                                | DEFAULT NULL            |
| `profile_image`  | `varchar(255)`                               | DEFAULT NULL            |
| `email`          | `varchar(30)`                                | DEFAULT NULL            |
